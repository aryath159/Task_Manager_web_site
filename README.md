# TaskFlow — Task Manager App

A simple task manager app built with plain HTML, CSS, and JavaScript.
This was my first complete frontend project, built while learning web development.

## Features
- Add tasks with title, description, priority, and due date
- Mark tasks as complete or pending
- Edit and delete tasks
- Filter tasks by status (All / Pending / Completed)
- Search tasks by title or description
- Progress bar showing overall completion
- Data saved in localStorage so tasks survive page refresh

## Tech Used
- HTML5 (semantic tags, forms, accessibility basics)
- CSS3 (flexbox, grid, animations, transitions, CSS variables)
- Vanilla JavaScript (DOM manipulation, localStorage, event delegation)

## Purpose
This project was built purely for learning and practice.
Every line was typed manually to revise concepts and build muscle memory.
